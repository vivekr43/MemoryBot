# Screenshots

This folder contains screenshots of the MemoryBot app used in the README.

## How to add screenshots

To capture and add the app screenshots, open your Terminal and run:

```bash
cd ~/Projects/MemoryBot
# Serve the app locally
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser, take a screenshot of:
- **main_ui.png** — the main chat interface
- **settings_panel.png** — the settings modal open

Save them to this `screenshots/` folder and push to GitHub.
